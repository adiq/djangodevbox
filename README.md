Django DEVBOX
=============
-------------

This project aims to deliver complete, easy-to-setup Vagrant box for developing Django Apps with Postgres.

## Currently: *UNUSABLE*
-------------


What we want to deliver?
-------------
  - Newset Django (1.6.X)
  - Postgres Database
  

  - Based on Debian
  - Running on Nginx
  - Running via uWSGI
  - Provisioning via Ansible
  

  - Super easy to setup
  - Keep this project up-to-date
